Pseudoclases principales de CSS3

Las Pseudoclases son propiedades CSS que se aplican a los elementos por medio de una palabra clave que especifican un estado especial.

:active

La pseudoclase :active especifica cuando un elemento está activo, es decir, cuando está siendo clicado por el usuario.



Podemos evidenciar que cuando presionamos el botón cambian algunas cosas y cuando soltamos el click vuelve a su estado original, esta modificación se logra gracias a la pseudoclase active.

:checked

La pseudoclase checked aplica a las etiquetas input de tipo radio y casillas de verificación, con esta podemos modificar las propiedades de cualquier casilla que el usuario selecione.






En este caso, cuando damos click sobre los elementos vemos que estos tienen un desplazamiento horizontal hacia la derecha y cuando lo quitamos vuelven a su posición inicial.

:default

Permite dar una apariencia diferente a la entrada de tipo radio o checkbox que aparezca marcada por defecto.




:disabled

Los estilos especificados en este selector se aplicaran a los campos que estén deshabilitados. En este caso el campo deshabilitado se mostrará de color negro.





:empty

Representa cualquier elemento que no tenga hijos. Los hijos pueden ser etiquetas o texto (incluido el espacio en blanco).


En este caso he creado un contenedor vacío y, a través de la pseudoclase :empty, le he añadido estilos.

:enabled

Representa cualquier elemento habilitado. Un elemento está habilitado si puede ser activado (es decir, seleccionado, se puede hacer click en él, acepta que se le introduzca texto, etc.) o si accepta el foco.





:first-child / :last-child

Representan el primer y último elemento entre un grupo de elementos hermanos, respectivamente.

  • Diferente (FIRST)
  • Normal
  • Normal
  • Normal
  • Normal
  • Diferente (LAST)

:focus

Representa un elemento (como una entrada de formulario) que ha recibido el foco. Generalmente se activa cuando el usuario hace clic o toca un elemento.





:hover

Coincide cuando el usuario interactúa con un elemento con un dispositivo señalador, pero no necesariamente lo activa. Generalmente se activa cuando el usuario se desplaza sobre un elemento con el cursor.

:valid

La pseudoclase :valid de CSS representa cualquier elemento de tipo input u otro elemento del formulario cuyo contenido se valide satisfactoriamente. Esto permite que los campos válidos adopten fácilmente una apariencia que ayuda al usuario a confirmar que sus datos están formateados correctamente.

:visited

La pseudoclase :visited de CSS representa enlaces que el usuario ya ha visitado.
